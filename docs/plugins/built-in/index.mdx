# Built-in Plugins

These plugins ship with BakShell in `plugins/` and are ready to use. Copy them to `~/.bshc/plugins/` and enable in `config.lua`.

| Plugin | Description | Entrypoints |
|--------|-------------|-------------|
| [Aquia Prompt](./aquia-prompt/) | Two-line prompt with git status and exit code (default) | `get_prompt`, `execute_command`, `set_exit_code` |
| [Git Prompt](./git-prompt/) | Git branch + status as a standalone prompt + `git-status` command | `get_prompt`, `execute_command` |
| [Autosuggest](./autosuggest/) | Fuzzy-matched command history + `suggest` command | `execute_command` |
| [Syntax Highlighting](./syntax-highlighting/) | Color-coded command tokens via `highlight` command | `execute_command` |
| [Powerlevel10k Prompt](./powerlevel10k/) | Full-featured p10k-style prompt with system monitoring | `get_prompt`, `execute_command` |

## Default config

```lua
plugins = {
    "aquia-prompt.lua",
    "autosuggest.lua",
    "syntax-highlighting.lua",
    "git-prompt.lua",
}
```
